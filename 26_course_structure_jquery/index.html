PICT B.E. Course Structure

Bachelor of Engineering (B.E.)

First Year (FE)

  • Engineering Mathematics-I
    Fundamentals of calculus, linear algebra, and differential equations for engineers.
  • Engineering Physics
    Concepts of mechanics, optics, and modern physics relevant to engineering.
  • Engineering Chemistry
    Chemical principles, materials science, and their engineering applications.
  • Basic Electrical Engineering
    Introduction to electrical circuits, machines, and basic electronics.
  • Programming in C
    Core programming concepts and problem-solving using the C language.

Second Year (SE)

  • Data Structures & Algorithms
    Study of data organization, searching, sorting, and algorithmic problem-solving.
  • Object Oriented Programming
    Principles of OOP, encapsulation, inheritance, and polymorphism (C++/Java).
  • Digital Logic Design
    Design and analysis of digital circuits, logic gates, and sequential circuits.
  • Discrete Mathematics
    Logic, sets, relations, graphs, combinatorics, and their applications in computing.
  • Computer Organization
    Architecture of computers, memory, CPU, and input/output systems.

Third Year (TE)

  • Database Management Systems
    Database models, SQL, normalization, transactions, and database design.
  • Operating Systems
    Concepts of process management, memory management, and file systems.
  • Computer Networks
    Networking fundamentals, protocols, OSI model, and network security basics.
  • Theory of Computation
    Automata, formal languages, Turing machines, and computability theory.
  • Software Engineering
    Software development life cycle, methodologies, and project management.

Final Year (BE)

  • Machine Learning
    Introduction to machine learning algorithms, supervised and unsupervised learning.
  • Cloud Computing
    Cloud models, virtualization, distributed systems, and cloud security.
  • Information & Cyber Security
    Principles of security, cryptography, network security, and ethical hacking.
  • Elective Subjects
    Advanced topics such as AI, IoT, Data Science, as per student interest.
  • Project Work
    Capstone project involving real-world problem solving and application development.
